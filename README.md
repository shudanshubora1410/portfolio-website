## 🌟 About

This is my personal developer portfolio website built to showcase my projects, skills, and achievements as a 3rd Year B.Tech IT student and aspiring Full-Stack Developer.

### Key Highlights

- 🎨 **Modern Design** - Clean, professional UI with gradient accents
- 📱 **Fully Responsive** - Works seamlessly on desktop, tablet, and mobile
- ⚡ **Performance Optimized** - Fast loading with smooth animations
- 🎯 **Project Focused** - Highlighting real-world projects including Hospital Management System
- 🔄 **Dynamic Content** - Typing animations, smooth scrolling, and interactive elements
